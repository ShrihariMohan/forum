# Forum app for Varivas

## Components

### Server

NodeJS express server

Steps to start

- Get G_SECRET and DB_URL from the mail.
- cd server
- npm install
- G_SECRET="<secret>" DB_URL="<url>" npm start

### client

Angular Server
Steps to start

- npm install
- ng serve
